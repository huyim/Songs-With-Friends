Songs, soon to be Songs with Friends

Users need a Spotify Premium account to use this app. Sorry, that's how it is. Fortunately Premium is free for three months, then $4.99/mo for students thereafter.

This app will only work if you, the user, have a playback device (phone, speaker, laptop browser window) already logged into your Spotify Premium account. For instance, first go to Spotify, log in, and go to the home screen so that the Web player is running. Play a bit of something so Spotify identifies the player as your most recently used device. Then click the button below. Don't click the button before you have a Spotify player ready to go.

To make this work nicer, your app should probably begin with an AJAX query to the server to get it to check with Spotify to see if there is an available playback device (eg. here or here).
